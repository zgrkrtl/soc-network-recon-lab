# SOC Network Recon Lab: Multi-Scenario Nmap Scans

This repository contains a series of **network reconnaissance labs** demonstrating different scanning techniques performed from a **Kali Linux attacker** to a **Windows 11 victim** VM. Each scenario includes:

- The **attack command(s)** used
- **Wireshark packet captures (PCAP)**
- **Analysis reports** explaining what happened on the network
- Optional **detection logic considerations** aligned with **MITRE ATT&CK** for SOC analysts

The purpose of this repo is to **practice SOC investigation skills**, **understand attacker behavior**, and **document findings professionally**
